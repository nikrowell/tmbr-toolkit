# TMBR Toolkit

A collection of JavaScript packages published under `@tmbr`

## Packages

| Package                        | Description                                  |
| ------------------------------ | -------------------------------------------- |
| [@tmbr/emitter](./emitter)     | Lightweight event emitter                    |
| [@tmbr/store](./store)         | State management built on emitter            |
| [@tmbr/utils](./utils)         | Miscellaneous utility functions              |
| [@tmbr/carousel](./carousel)   | Embla Carousel extension with added features |
| [@tmbr/shader](./shader)       | Lightweight WebGL shader class               |
| [@tmbr/component](./component) | Reactive DOM component base class            |
